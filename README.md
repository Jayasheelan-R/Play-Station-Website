# Play-Station-Website

A simple front-end learning project that recreates parts of a PlayStation-style website for practice and education.

## About

This repository contains a small static website built with HTML and CSS. It's intended as a learning project to practice layout, responsive design, and basic front-end skills.

Important: this project is strictly educational and NOT affiliated with or endorsed by Sony Interactive Entertainment or any PlayStation entities. All PlayStation names, logos, images, and other trademarks referenced here are the property of their respective owners and are used only for non-commercial learning/demonstration purposes. No copyright infringement is intended.

## Files

Files in this repository:

- `main.html` — Main landing page
- `psconsole.html` — Console page
- `psgames.html` — Games listing page
- `psplus.html` — PlayStation Plus page
- `psvr.html` — PSVR page
- `accs.html` — Accessories page
- `style.css` — Main stylesheet
- `Images/` — Image assets used by the site

## How to run

1. Clone or download the repository.
2. Open any `.html` file in a modern browser (Chrome, Firefox, Safari) — no server required for local development.

Example: open `main.html` in your browser to view the site.
